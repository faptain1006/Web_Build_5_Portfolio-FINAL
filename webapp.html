Deep Packet Inspector

Deep Packet Inspector is a network security tool that performs in-depth analysis of network traffic at the packet level. It helps identify and inspect the content of data packets, allowing for the detection of anomalies, security threats, and protocol violations within a network.

  • Network Protocols: TCP/IP, UDP, ICMP
  • Programming: Python, C/C++
  • Cloud Compatibility: AWS, Azure, Google Cloud
  • Machine Learning: TensorFlow, PyTorch

Honey-Trap

Honey-Trap is a network honeypot application designed to emulate vulnerable systems and services, attracting and detecting malicious activities. It serves as a decoy to divert and analyze potential threats, providing valuable insights into the tactics and techniques used by attackers.

Tech Stack:

  • Honeypot Framework: Honeyd, Cowrie, Dionaea
  • Networking: iptables, TCP/IP
  • Programming: Python, C/C++
  • Logging and Analysis: ELK Stack (Elasticsearch, Logstash, Kibana)
